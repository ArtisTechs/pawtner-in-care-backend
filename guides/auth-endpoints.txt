AUTH ENDPOINTS

Base URL:
http://localhost:8080/api/auth

1. Sign Up
POST /signup
Content-Type: application/json

Sample body:
{
  "firstName": "Jane",
  "middleName": "Marie",
  "lastName": "Doe",
  "email": "jane@example.com",
  "password": "Test1234!"
}

Notes:
- Role defaults to USER
- User ID is auto-generated as UUID
- Email must be unique

2. Login
POST /login
Content-Type: application/json

Sample body:
{
  "email": "jane@example.com",
  "password": "Test1234!"
}

Sample success response:
{
  "tokenType": "Bearer",
  "accessToken": "your-token-here",
  "user": {
    "id": "generated-uuid",
    "firstName": "Jane",
    "middleName": "Marie",
    "lastName": "Doe",
    "email": "jane@example.com",
    "role": "USER"
  }
}

3. Send OTP
POST /send-otp
Content-Type: application/json

Authorization:
No bearer token required

Sample body:
{
  "email": "jane@example.com",
  "purpose": "signup"
}

Allowed purpose values:
- signup
- login
- reset-password

4. Confirm OTP
POST /confirm-otp
Content-Type: application/json

Authorization:
No bearer token required

Sample body:
{
  "email": "jane@example.com",
  "otp": "123456",
  "purpose": "signup"
}
